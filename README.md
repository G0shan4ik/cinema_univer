# 🎬 CinemaHub

## 📌 Описание

CinemaHub — это сайт кинотеатра, где можно посмотреть афишу, выбрать фильм и забронировать билет онлайн. Пользователь может просматривать информацию о фильмах, доступные сеансы и выбирать места в зале. Также доступен личный кабинет с историей билетов и избранными фильмами.

---

## 👥 Для кого этот сайт

- Посетители кинотеатра
- Любители кино и новинок
- Пользователи онлайн-сервисов покупки билетов
- Администраторы кинотеатра

---

## 🚀 Основные возможности

- Просмотр фильмов и афиши
- Получение информации о фильмах
- Просмотр сеансов
- Бронирование билетов
- Личный кабинет пользователя
- Избранные фильмы
- Поиск фильмов
- Админ панель

---

## 🧩 Страницы сайта

- `/` — главная страница (афиша, новинки)
- `/auth` — регистрация и вход
- `/movies` — список фильмов
- `/movies/:id` — страница фильма
- `/booking` — бронирование мест
- `/profile` — личный кабинет
- `/profile/tickets` — билеты пользователя
- `/favorites` — избранные фильмы
- `/search` — поиск
- `/admin` — админ-панель

## 🖼️ Описание страниц

- Отображение популярных фильмов

---

- Список фильмов

👉 `GET /api/movies`

---

### Страница фильма (`/movies/:id`)
- Постер, описание

👉 `GET /api/movies/:id`

---

### Бронирование (`/booking`)
- Схема зала
- Выбор мест  
👉 данные приходят из API (sessions + tickets)

### Профиль (`/profile`)
- Данные пользователя
- Переход к билетам и избранному

---

## 🗄️ Структура данных

### User
- id
- email (уникальный)
- name
- password_hash
- keyword_hash (для восстановления пароля)
- created_at
- role (user / admin)
- is_active

---

### Movie
- id
- title
- description
- genre
- duration_minutes
- rating
- poster_url
- release_date

---

### Hall
- id
- name
- total_rows
- seats_per_row

---

### Session
- id
- movie_id (FK → Movie)
- hall_id (FK → Hall)
- start_time
- price

---

### Ticket
- id
- user_id (FK → User)
- session_id (FK → Session)
- seat_row
- seat_number
- status (booked / paid / cancelled)
- created_at

👉 Ограничение:
- уникальность места на сеанс  
(`session_id + seat_row + seat_number`)

---

### Favorite
- id
- user_id (FK → User)
- movie_id (FK → Movie)

👉 Ограничение:
- один фильм можно добавить в избранное только один раз

---

## 🔌 API

### Аутентификация
- `POST /auth/register` — регистрация
- `POST /auth/login` — вход

---

### Фильмы
- `GET /movies` — список фильмов
- `GET /movies/:id` — информация о фильме

---

### Билеты
- `POST /tickets` — бронирование билета

---

## 📌 Примечание

Связи между сущностями реализованы через внешние ключи (ForeignKey), что обеспечивает целостность данных и корректную работу системы бронирования.
